# BasicEasyStat

一个**纯前端、零依赖的描述统计工具**。打开 `index.html` 即可使用,无需安装、无需联网,适合实验室、工程现场、课堂上快速做一组数据的统计描述。

> 双击 `index.html` 就能跑,数据全部在浏览器本地处理。

---

## ✨ 主要功能

- **多种数据输入方式**
  - 手动输入 / 粘贴数字(空格、逗号、分号、中文逗号自动识别)
  - **OCR 截图识别**:从 Excel 截图、仪器读数、纸质表格中识别数字(浏览器本地完成,不上传服务器)
  - 支持点击、拖拽、`Ctrl+V` 三种方式喂图

- **完整的描述统计指标**

  | 类别 | 指标 |
  |------|------|
  | 样本 | 样本数 n、Σx、Σx² |
  | 集中趋势 | 均值、中位数、众数(支持多峰提示) |
  | 离散程度 | RSD 变异系数、标准差 s、方差 s²、标准误 SE、极差 R、IQR、Q1/Q3 |
  | 分布形状 | 偏度(Skewness)、峰度(Excess Kurtosis) |
  | 极值 | min、max |
  | 总体 | 总体标准差 σ、总体方差 σ² |

- **多种导出格式**:一键复制为 TSV(直接粘到 Excel)、Markdown 表格、JSON

- **响应式 UI**:桌面 / 平板 / 手机自适应

---

## 🚀 使用方法

### 方式一:直接打开(最简单)

双击 `index.html`,在浏览器里就能用。

### 方式二:本地起个静态服务(避免部分浏览器 file:// 限制)

```bash
# 进入项目目录
cd BasicEasyStat

# Python 3
python -m http.server 8000

# 或者 Node.js
npx serve .
```

然后浏览器打开 `http://localhost:8000`。

### 操作步骤

1. 把数据粘贴到输入框(支持 `1 2 3`、`1,2,3`、`1;2;3`、`1,2,3` 等格式)
2. 统计结果实时显示
3. 右上角按钮可复制为 **Excel / Markdown / JSON**
4. 点 **OCR 识别** 从截图里提取数字

## 🔢 输入格式示例

```
98.2 99.1 98.8 99.5 98.7 99.0 98.5 99.3 98.8 99.2
```

也支持:

```
98.2, 99.1, 98.8, 99.5
98.2;99.1;98.8;99.5
98.2，99.1，98.8，99.5
```

## 📤 导出格式示例

### TSV / Excel

直接粘到 Excel,自动按列对齐。

```
指标	值
样本数 n	10
总和 Σx	989.1
...
```

### Markdown

```markdown
| 指标 | 值 |
|---|---|
| 样本数 n | 10 |
| 均值 x̄ | 98.91 |
...
```

### JSON

```json
{
  "n": 10,
  "data": [98.2, 99.1, 98.8, ...],
  "均值 x̄": 98.91,
  ...
}
```

## 🔒 隐私

- 数据只在浏览器内存中计算,**不会发送到任何服务器**
- OCR 使用 [Tesseract.js](https://github.com/naptha/tesseract.js) 在 WebAssembly 里本地运行,图片不上传
- 唯一需要联网的环节是首次加载 Tesseract.js 的引擎文件(约 10MB,之后会被浏览器缓存)

---

## 🛠️ 技术栈

- **HTML + CSS + 原生 JavaScript** —— 一个文件搞定
- [Tesseract.js v5](https://github.com/naptha/tesseract.js) —— OCR 引擎
- 无任何构建步骤、无依赖
